# Cultural Adaptation Tracker - SQL Project

## Overview

This SQL project documents and analyzes my cultural adaptation journey as a Brazilian living in Madrid, tracking how habits, social networks, and identity evolve when moving to a new country.

## Database Structure

The database consists of these tables:
- `Users`: Personal information
- `CulturalEncounters`: First experiences with cultural elements
- `CulturalAdaptations`: Changes in habits
- `SocialNetworkEvolution`: Development of relationships
- `IdentityEvolution`: Personal changes and self-perception

## SQL Skills Demonstrated

- Relational database design with foreign keys
- Complex SQL queries using joins and subqueries
- Analysis of temporal patterns
- Quantitative tracking of qualitative experiences
